# Scraping
Extract Quora questions in specific topic

Scraping tools: Selenium, beautiful soup

File needed:
- topic.txt 
  - text file with list of topics seperated in lines

Features extract:
- Question information:
  - Title
  - Creator
  - Answer count
  - Topic list
  - Create date
  - Followers
  - Views
- Answer information:
  - Content
  - Responder
  - Create date
  - No. of upvote
  - View
